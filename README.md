Sistema de Gestión Académica (Desktop)
Este es un software de escritorio desarrollado en Java Swing diseñado para optimizar el control de alumnos, la gestión de calificaciones y la administración de usuarios en instituciones educativas.

Características Principales
Registro Seguro de Usuarios: Sistema con niveles de acceso diferenciados para Administradores y Docentes.

Validación de Roles: Los nuevos administradores requieren una clave de autorización especial para completar su registro.

Gestión de Alumnos: Visualización detallada de la información del estudiante, incluyendo nombre, grado y teléfono.

Control de Calificaciones: Registro y actualización de tareas o actividades por materia.

Cálculo Automático: El sistema calcula el promedio de notas y asigna un estatus de Aprobado o Reprobado en tiempo real.

Reportes PDF: Generación de documentos oficiales con el historial académico (módulo ya integrado).

Tecnologías Utilizadas
Lenguaje: Java (JDK 17+)

Interfaz Gráfica: Java Swing (NetBeans IDE).

Base de Datos: SQLite para una portabilidad total.

Librerías: JDBC para la conexión a base de datos.
